My Projects

Music Organizer – Playlist Sorting App

Background: Built with Next.js, Tailwind CSS, and TypeScript, this project allows users to sort playlists by artist, album, title, or year. It stemmed from limitations I experienced with Spotify’s web player.

Problem: Large music libraries can be hard to manage. This project simplifies organization by providing a custom front-end sorting experience.

Process: Designed as an e-commerce mockup with clean UI/UX, this ongoing project started with frontend logic. I plan to add backend features and enhancements like BPM, mood filters, and animations.

Results: A working mockup showcasing my frontend skills, responsive layout, and a scalable design approach for real-world music platforms.

SilentType – E-commerce Website for Linear Keyboards

Background: Created for a JavaScript class project, this e-commerce site features a total cost calculator. I focused on the user interface and effective product display.

Problem: Many beginner e-commerce sites lack UI polish and interactivity. This project aimed to solve both with a clean product layout and responsive calculator logic.

Process: Built with HTML, CSS, and JavaScript, I concentrated on layout aesthetics, form handling, and product section clarity. Design principles like spacing, hierarchy, and user flow were key.

Results: Delivered a sleek e-commerce interface that balances simplicity and functionality—ideal for showcasing modern tech products like keyboards.

Minimal Sneaker Photo – Rule of Thirds Composition

Background: A visual composition project using Photoshop to create an aesthetic sneaker photo, applying the rule of thirds to emphasize product design.

Problem: Product photography can feel flat without visual structure. This project tackled that by using space and layout intentionally.

Process: Positioned the sneaker off-center on a clean background. Adjusted lighting in Photoshop to add depth while maintaining a minimal style.

Results: A clean, engaging product photo ideal for portfolios or e-commerce. Demonstrates my growing design eye and photo-editing skills.
